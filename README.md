# AI-Study-Planner
AI Study Planner

Project Overview

AI‑powered study planner that generates personalized study schedules based on subjects, difficulty, and remaining exam days. This beginner-level project demonstrates practical application of AI concepts to improve study planning efficiency.

⸻

Technologies Used
	•	Python
	•	AI logic / simple algorithms for scheduling
	•	pandas & numpy (for data management)
	•	matplotlib / seaborn (optional visualization of study plans)
